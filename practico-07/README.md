# Practico 07 - JSON y MongoDB

En este practico se uso el proyecto Java provisto por la catedra para trabajar JSON,
y despues se paso al modelado/consultas en MongoDB.

## Material que se agrego

Se copiaron estos archivos al repo:

- `recursos/gson-2.8.6_145823522906.jar`
- `recursos/proyectoJsonJava_145823133931.rar`

El proyecto del `.rar` se descomprimio en:

- `recursos/json/`

Y dentro de ese proyecto quedaron agregadas estas libs:

- `recursos/json/lib/gson-2.8.6.jar`
- `recursos/json/lib/mysql-connector-j-8.0.33.jar`

## Que hay en cada parte

- `ejercicio-01/README.md`: consultas sobre `jsonInventario.txt` con JsonPath.
- `ejercicio-02/README.md`: programa Java (Gson + JDBC) que genera un JSON unico de facturas con sus items.
- `ejercicio-03/README.md`: diseno de base y colecciones en MongoDB para el modelo del enunciado.
- `ejercicio-04/README.md`: carga minima pedida (2 usuarios, 4 noticias y 2 comentarios por noticia).
- `ejercicio-05/README.md`: consultas MongoDB de los incisos a, b y c.
- `recursos/json/src/jsonPath/Ejercicio1Inventario.java`: codigo del ejercicio 1.
- `recursos/json/src/jsonPath/Ejercicio2FacturaJson.java`: codigo del ejercicio 2.
