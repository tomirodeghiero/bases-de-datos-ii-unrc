# Practico 04 - JDBC

Resolucion del Practico 4 usando API JDBC de Java.

## Estructura

- `ejercicio-01/`: programa por consola para alta y listado de clientes (inciso 1.a de Practico 1).
- `ejercicio-02/`: programa por consola que invoca la funcion de maxima cotizacion del Practico 3 (ejercicio 2) y muestra el parametro de salida.
- `ejercicio-03/`: programa por consola que lista metadatos de tablas (columnas, PK, UQ y FK) del modelo del inciso 1.c de Practico 1 para PostgreSQL y Oracle.
